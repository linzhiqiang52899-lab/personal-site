# 欢迎使用个人内容托管网站

这是一个简单快速的内容托管解决方案，支持移动端浏览。

## 如何使用

### 添加新内容

只需要将 `.md` 或 `.html` 文件放入 `content` 目录，系统会自动：

1. 扫描到新文件
2. 生成访问链接
3. 在首页显示

### 访问链接

文件名即为访问路径，例如：

- 文件 `content/关于我.md` → 访问 `http://localhost:3000/关于我`
- 文件 `content/project.html` → 访问 `http://localhost:3000/project`

## 功能特点

- ✅ 自动扫描和监听文件变化
- ✅ Markdown 自动渲染为 HTML
- ✅ 移动端响应式设计
- ✅ 代码高亮支持
- ✅ 一键分享链接
